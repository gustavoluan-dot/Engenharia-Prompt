# 🚀 Engenharia de Prompt (Prompt Engineering)

Olá! Bem-vindo ao meu repositório sobre **Engenharia de Prompt**. 

Sabe quando você conversa com uma Inteligência Artificial e precisa dar as instruções certinhas para ela te dar a melhor resposta possível? Isso é a Engenharia de Prompt! É como aprender a "falar a língua dos robôs" de um jeito super inteligente e eficaz.

---

## 📂 O que tem aqui dentro?

Neste projeto, eu guardo os meus testes, fórmulas secretas e os melhores comandos que criei:

* **`engenharia de prompt` / `engenharia-de-prompt`**: Pastas com exemplos práticos, truques e guias para conversar com a IA.
* **Exemplos Reais**: Técnicas para fazer a IA agir como um professor, um programador ou qualquer profissão!

---

## 🧠 O que eu estou aprendendo?

* A estruturar perguntas perfeitas para não deixar a IA confusa.
* A usar técnicas avançadas (como dar exemplos para a IA antes de ela responder).
* A melhorar resultados para tarefas do dia a dia.

---

## 🛠️ Como usar este projeto?

1. Entre nas pastas ali em cima para ver os arquivos.
2. Copie os comandos (prompts) que eu deixei anotados.
3. Cole na sua IA favorita (como o ChatGPT) e veja a mágica acontecer!
